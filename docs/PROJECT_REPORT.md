
# **Project Report: Shoji Amida Generator Upgrade**

## **1. 概要**
本プロジェクトでは、FreeCAD を用いた「あみだくじ型障子（Shoji Amida）」のパラメトリック・ジェネレーターを大幅にアップグレードしました。従来のロジックのみの生成から、対話的な UI 操作、および実務的なプレカット連携までをカバーする統合ツールへと進化させました。

## **2. 主要なアップデート項目**

### **🎨 ユーザーインターフェース (UI) の刷新**
- **専用ツールバーの導入**: FreeCAD 上部に直感的な操作が可能なボタン群（パラメーター設定、再生成、各種エクスポート）を追加。
- **対話的ダイアログ**: スライダーとスピンボックスを用いて、障子のサイズ、繁（しげ）の数、あみだパターンの密度をリアルタイムに調整可能。

### **🏭 実務連携・エクスポート機能**
- **CEDXM (Member-based) 出力**: プレカット業界標準（CEDAR XML）に基づき、各部材（土台、梁、管柱、垂木）を属性付きで出力。仕様書に基づき、`mokuzai`, `syurui`, `w`, `h` などのプロパティを完全実装。
- **Blender ダイレクト連携**: OBJ 形式での書き出しと同時に Blender を自動起動し、即座にレンダリングや詳細モデリングへ移行可能。
- **3D Web ビューア生成**: GLTF 出力と共に、ブラウザで閲覧可能な HTML ビューア（`<model-viewer>` 使用）を自動生成。
- **2D/3D 汎用形式**: PNG（スクリーンショット）、DXF（2D CAD）の出力に対応。

### **⚙️ 技術的改善 (Robustness)**
- **FreeCAD 1.0 への最適化**: `Mesh` モジュールを活用した安定したテッセレーションと書き出し処理の実装。
- **エラーハンドリングの強化**: モジュール不在時のフォールバック処理や、Python 3.12 未満の環境での SyntaxError 回避（f-string の修正など）。
