# ADR 003: CEDXM 規格による木材プレカットデータ連携

## 状況
生成した障子のパラメトリック・モデルを、実際の木材加工（プレカット）工程で利用可能なデータとして出力する必要がある。汎用的な CAD 形式だけでなく、日本の木材プレカット業界の標準規格である CEDXM (CEDAR XML) 形式をサポートすることが求められた。

## 決定
生成される個々の部材 (Member) の属性情報を CEDXM 形式で出力する。

具体的には、以下の属性を各 Member にマッピングする：
- **Pset_param**: `mokuzai`, `syurui`, `jyusyu`, `toukyuu`, `kesyou`, `w`, `h`
- **Geometry**: `Start`, `End` (3D 座標), `Length` (部材長)

部材種別 (syurui) の対応：
- 上枠 → `hari`
- 下枠 → `dodai`
- 縦繁/縦枠 → `kuda`
- 横繁 → `taruki`

## 影響
- パラメトリック・モデルを単なる視覚的な CAD データとしてではなく、製造に直結する生産データとして活用可能になった。
- 外部のプレカット CAD や木材加工機とのシームレスな連携が可能になった。
