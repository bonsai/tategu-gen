# ADR 001: ツールバーの生存期間管理 (GUI オブジェクトのガベージコレクション回避)

## 状況
FreeCAD 上で Python スクリプトを実行してツールバーを表示させた際、スクリプトの実行が終了すると、作成したツールバーオブジェクトが Python のガベージコレクション (GC) によって破棄され、画面から消えてしまう問題が発生した。

## 決定
ツールバーオブジェクトを FreeCAD のグローバルな GUI オブジェクトである `Gui` モジュールに明示的にアタッチし、参照を維持する。

具体的には、以下の実装を行う：
- `Gui._timber_toolbars` というリストを `Gui` モジュールに追加する。
- 新しく作成したツールバーのインスタンスをこのリストに格納する。
- 再生成時には、既存のツールバーを特定して削除 (`removeToolBar` および `deleteLater`) し、リストを更新する。

## 影響
- スクリプト実行終了後も、ユーザーが FreeCAD を閉じない限りツールバーが表示され続ける。
- 重複してツールバーが表示されることを防ぎ、常に最新の状態を維持できる。
