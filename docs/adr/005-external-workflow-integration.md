# ADR 005: 外部ビジュアライゼーション・ワークフロー連携 (Blender/Web)

## 状況
FreeCAD 上での設計だけでなく、リアルなレンダリングやウェブ上での 3D プレビューなど、設計した障子を外部ツールで活用する需要が高い。しかし、手動でのエクスポートや変換は手間がかかり、作業フローの断絶を招く。

## 決定
FreeCAD の外部ツールとの連携を自動化し、ボタン一つで外部ワークフローへ接続可能にする。

具体的には、以下の機能を実装する：
- **Blender 連携**: OBJ 形式での書き出しと同時に、ローカルマシンの Blender を自動的に起動してファイルを開く。
- **3D Web ビューア**: GLTF 形式での書き出しと同時に、Google の `<model-viewer>` を使用した HTML ファイルを自動生成し、ブラウザでプレビューを表示する。

## 影響
- 設計からビジュアライゼーションまでのリードタイムが大幅に短縮された。
- 非 CAD ユーザー（施主や設計士）への 3D モデルの共有が、ウェブブラウザを通じて容易になった。
- 高度なレンダリングや詳細なモデリングへの移行がスムーズになった。
